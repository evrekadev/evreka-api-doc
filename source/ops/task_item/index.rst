   +-----------+-------------------------------------+----------------------------------------------------+
   | Method    | Endpoint                            | Description                                        |
   +===========+=====================================+====================================================+
   | POST      | /task_items                         | Create a task item with given parameters           |
   +-----------+-------------------------------------+----------------------------------------------------+
   | GET       | /task_items                         | List the task items of the authenticated client    |
